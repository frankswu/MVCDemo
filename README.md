MVCDemo
=======

一个 iOS Demo，展示如何用 MVC 的方式构建程序。基本功能是访问一个StartupNews 的一个非官方 API，加载列表；点击列表后进入相应的网页。用到了 AFNetwork，网页展示用了 Readability。

*感谢@[xhinking](https://github.com/xhinking/StartupNewsAPI)提供的 API！*
